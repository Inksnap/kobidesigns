Design Guidelines

Follow these guidelines to ensure the best print quality

File Format

Use PDF with embedded fonts for best results. AI and PSD files are also accepted.

Color Mode

Use CMYK color mode for printing. RGB files will be converted and may have color shifts.

Bleed Area

Include 3mm bleed on all sides. Important content should be 3mm from the edge.

Resolution

Use 300 DPI minimum for all images and graphics to ensure crisp print quality.
